개인정보처리방침

시행일: 2026년 4월 13일

Tedypark("회사")는 「개인정보 보호법」 및 관련 법령에 따라 이용자의 개인정보를 보호하고, 이와 관련한 고충을 신속하게 처리하기 위하여 다음과 같이 개인정보처리방침을 수립·공개합니다.

제1조 수집하는 개인정보 항목 및 수집 방법

1. 수집 항목

분류수집 항목수집 근거
계정 정보이메일 주소, 인증 토큰서비스 가입 및 로그인

중요: 사진·동영상 원본 파일은 회사 서버에 저장되지 않습니다. AI 분석을 위해 기기에서 Supabase Edge Function(회사 백엔드)으로 전송하고, Edge Function이 Google Gemini API에 분석을 요청합니다. 분석 완료 즉시 원본 미디어는 삭제되며, 분석 결과만 데이터베이스에 저장됩니다. Google의 API 데이터 처리 정책은 Google AI API 이용약관을 따릅니다.

2. iCloud 공유 보관함

오늘아이는 반려자·가족 구성원이 공유한 기록을 하루 일기에 통합하기 위해 Apple iCloud 공유 보관함에 있는 사진·동영상도 읽어 분석합니다.

  • iCloud 공유 보관함은 Apple Inc.이 운영하는 서비스이며, 회사가 해당 파일을 별도 서버에 저장하지 않습니다.
  • 공유 보관함에서 읽은 사진·동영상도 동일한 분석 경로(Supabase Edge Function → Gemini API)를 따르며, 원본은 분석 후 즉시 삭제됩니다.
  • 공유 보관함 접근 권한은 기기의 사진 라이브러리 권한 설정에서 관리됩니다.

3. 수집 방법

  • 이용자가 앱에 직접 입력
  • 앱 서비스 이용 과정에서 자동 수집
  • 기기의 사진 라이브러리 접근 권한 허용 시 (원본은 기기 로컬에만 존재, 로컬 식별자·메타데이터만 서버에 저장)

제6조 정보주체의 권리 및 행사 방법

이용자는 다음 권리를 행사할 수 있습니다.

  • 열람 요구: 본인의 개인정보 처리 현황 확인
  • 정정·삭제 요구: 잘못된 정보 수정 또는 삭제 요청
  • 처리 정지 요구: 개인정보 처리 일시 정지 요청
  • 이의 제기: 개인정보 처리에 대한 이의 제기

행사 방법: 앱 내 설정 화면 또는 support@tedypark.com으로 요청
처리 기간: 요청 접수 후 10일 이내 처리

만 14세 미만 아동의 경우 법정대리인이 권리를 행사할 수 있습니다.

제7조 개인정보의 안전성 확보 조치

  • 암호화: 비밀번호 및 민감 정보 암호화 저장
  • 접근 제한: Row Level Security(RLS)를 통한 사용자별 데이터 접근 통제
  • 전송 보안: HTTPS/TLS 암호화 전송
  • 키 관리: API 키는 디바이스 Keychain에 암호화 저장, 서버에 평문 저장하지 않음
  • 미디어 미보관: 사진·동영상 원본을 서버에 저장하지 않아 데이터 유출 위험 최소화
  • 접근 로그: 비정상 접근 탐지를 위한 로그 관리

제8조 자동 수집 장치의 설치·운영 및 거부

서비스 품질 개선을 위해 앱 내 최소한의 사용 통계를 수집합니다. 이용자는 기기의 앱 설정에서 사진 라이브러리 접근 권한을 차단하여 미디어 분석 기능의 자동 수집을 거부할 수 있습니다. 단, 거부 시 해당 기능의 이용이 제한됩니다.
